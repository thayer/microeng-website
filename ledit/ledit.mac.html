L-Edit for the Macintosh

MACINTOSH KEYSERVER FILES at Dartmouth:

  • (6 keys) L-Edit folder: L-Edit, ledit.tdb, Supp L-Edit Install (includes samples)

  • (2 keys) L-Edit/DRC folder: L-Edit, ledit.tdb, Supp L-Edit/DRC Install (includes samples)

  • (1 key) L-Edit/Extract folder: L-Edit, ledit.tdb, Supp L-Edit/Extract Install (includes samples)

  • Micromachining MCNC/MUMPS folder: mpmems.tdb, mpmems.xst (technology files for MCNC's multi-user micromachining run. Move these files to your L-Edit folder, and rename mpmens.tdb to ledit.tdb if you plan to do micromachining design rather than VLSI design.)

WHICH MACINTOSH FILES SHOULD I DOWNLOAD AND RUN?

Start with just the L-Edit folder, and the micromachining and CircCIF folders if you plan to do micromachining design. Come back for the L-Edit/DRC or L-Edit/Extract when you know you need them. We have six keys to L-Edit, two to L-Edit/DRC, and one to L-Edit/Extract. While you can download all files, unless you are planning to immediately use the DRC or Extract features, please run only L-Edit. This leaves the few L-Edit/DRC and L-Edit/Extract keys for those who really need them.

For more information about L-Edit, or to borrow an L-Edit manual, contact Chris Levey (c.levey@dartmouth.edu).
